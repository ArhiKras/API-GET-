# HTTP Client и Информация о Странах

Проект для тестирования HTTP-запросов и получения информации о странах через различные API.

## Возможности

- **GET и POST запросы** - выполнение произвольных HTTP-запросов с настраиваемыми параметрами и заголовками
- **Информация о странах** - получение полной или краткой информации о любой стране мира
- **Случайные изображения собак** - получение ссылок на случайные изображения собак через API

## Структура проекта

```
VPd01/
├── main.py                 # Основной модуль с CLI меню

├── short_country_info.py  # Модуль для краткой информации о странах

└── README.md              # Документация
```

## Установка

### 1. Клонирование репозитория

```bash

```

```bash
pip install -r requirements.txt
```

## Использование

### Запуск основного приложения

```bash
python main.py
```

### Главное меню

После запуска вы увидите меню с опциями:

1. **GET запрос** - выполнение GET запроса к указанному URL
2. **POST запрос** - выполнение POST запроса с данными
3. **Информация о стране** - подменю для работы со странами
4. **Случайная собака** - получение ссылки на случайное изображение собаки
5. **Выход** - завершение работы программы

### Работа со странами

При выборе пункта "3" откроется подменю:

1. **Полная информация по стране** - детальная информация с географическими, демографическими, политическими и экономическими данными
2. **Краткая информация по стране** - основные данные: столица, население, валюта
3. **Назад в главное меню** - возврат в основное меню

### Примеры использования

#### GET запрос
```
Выберите тип запроса: 1
Введите URL: https://api.github.com/users/octocat
Параметры запроса: {} (или Enter для пропуска)
Заголовки: {} (или Enter для пропуска)
```

#### Информация о стране
```
Выберите тип запроса: 3
Выберите тип информации: 1 (или 2 для краткой)
Введите название страны: Norway
```

## Зависимости

- **requests** - библиотека для HTTP-запросов
- **colorama** - библиотека для цветного вывода в консоль (Windows)

## Модули

### http_client.py

Модуль для выполнения HTTP GET запросов с базовой проверкой статуса.

**Функции:**
- `get(url, params=None, timeout=10, headers=None)` - выполнение GET запроса с проверкой статуса

### country_info.py

Модуль для получения и отображения полной информации о стране.

**Функции:**
- `get_country_info(country)` - получение данных о стране через API
- `display_country_info(country_data)` - красивое отображение информации с цветами

**Отображаемая информация:**
- Официальное название, регион, континент
- Географические данные (координаты, площадь, границы)
- Демографические данные (население, коэффициент Джини)
- Политическая информация (столица, независимость, членство в ООН)
- Экономическая информация (валюта)
- Культурная информация (языки)
- Коды и идентификаторы
- Ссылки на карты

### short_country_info.py

Модуль для получения и отображения краткой информации о стране.

**Функции:**
- `get_country_info(country)` - получение данных о стране через API
- `display_short_country_info(country_data)` - краткое отображение ключевых полей

**Отображаемая информация:**
- Название страны
- Столица
- Население
- Валюта

## API Endpoints

Проект использует следующие публичные API:

- **REST Countries API** - `https://restcountries.com/v3.1/name/{country}` - информация о странах
- **Dog CEO API** - `https://dog.ceo/api/breeds/image/random` - случайные изображения собак
